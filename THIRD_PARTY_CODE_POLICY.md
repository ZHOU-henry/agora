# Third-Party Code Policy

## Rules

1. Do not copy code from repositories or websites without a clear license.
2. Prefer Apache-2.0, MIT, BSD, or ISC licensed inputs.
3. Preserve attribution and license notices for reused code.
4. Record provenance for any imported module, snippet, or asset.
5. Treat GPL/AGPL/LGPL inputs as a separate review decision before use.

Open-source intent and delayed monetization do not remove license obligations.
