# Agora v0 Scope

Agora v0 is an **AI Agent Platform MVP**, not a full marketplace.

## In Scope

- curated agent catalog
- task intake and routing
- execution record
- operator review
- lightweight trust and provenance signals

## Out Of Scope

- payments
- revenue sharing
- tokenization
- trusted execution hardware
- enterprise tenancy complexity
- full marketplace liquidity mechanics

## Product Principle

First make agent usage understandable and auditable.

Only then consider marketplace expansion.
