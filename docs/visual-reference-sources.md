# Visual Reference Sources

These sources were used as reference material for Agora's industrial deployment visuals.

Rule:

- reference only
- do not copy third-party images, gifs, or diagrams into the repo unless license and attribution obligations are clearly satisfied
- local assets in `apps/web/public/media/` were generated from scratch after studying these references

## Official Product / Company References

- IUNA Assembly Inspector
  Link: https://iuna.ai/assembly_inspector/
  Why it mattered: showed a robot-mounted industrial camera and inspection cell framing that feels close to real line deployment rather than abstract AI branding.

- Overview AI
  Link: https://www.overview.ai/
  Why it mattered: reinforced the browser-based industrial inspection console pattern with clear OK / NG framing and production-floor monitoring posture.

- Rockwell Automation VisionAI
  Link: https://www.rockwellautomation.com/en-us/products/software/factorytalk/analytics-logixai/visionai.html
  Why it mattered: anchored the OT / PLC / factory-control side of how industrial AI systems are positioned in real operations.

- Brightpick Autopicker
  Link: https://brightpick.ai/order-picking/
  Why it mattered: grounded the warehouse scene in a real aisle-picking robot posture where WMS tasking, tote retrieval, and pack-station closure all belong to one deployment.

- ANYmal Autonomous Robotic Inspection
  Link: https://www.anybotics.com/robotics/anymal/
  Why it mattered: gave a strong real-world reference for inspection robots carrying visual and thermal sensors in industrial maintenance environments.

- Siemens Senseye Predictive Maintenance with Generative AI
  Link: https://news.siemens.com/en-us/siemens-elevates-predictive-maintenance-with-innovative-generative-ai/
  Why it mattered: reinforced that maintenance agents are not just anomaly detectors; they are part of the maintenance decision and action loop.

## GitHub Demo / Prototype References

- Stepper Robot Arm
  Link: https://github.com/RonaldChicche/Stepper-Robot-Arm
  Why it mattered: useful as a motion and joint-behavior reference for a simple robot-arm animation language.

- Automated Industrial Inspection System
  Link: https://github.com/Gouhari/Automated-Industrial-Inspection-System
  Why it mattered: useful as a reference for browser-based defect inspection UI framing and industrial quality workflow storytelling.

- RobotecAI ROSCon2023Demo
  Link: https://github.com/RobotecAI/ROSCon2023Demo
  Why it mattered: strong reference for multi-robot warehouse simulation, fulfillment-center layout, and how robotic warehouse scenes can be staged legibly.

- warehouse-robot
  Link: https://github.com/abel-gr/warehouse-robot
  Why it mattered: useful as a reference for the coupling between warehouse robot behavior and a web control interface.

- AI-Powered Predictive Maintenance System for Industrial Robotics
  Link: https://github.com/melisasvr/AI-Powered-Predictive-Maintenance-System-for-Industrial-Robotics
  Why it mattered: useful reference for maintenance dashboard language, robot-health framing, and the sensor-to-maintenance-task loop.
